# zbiolock

Secure biometric authentication for Capacitor apps with Fingerprint, Face ID, Android Face Unlock, Device PIN fallback, and secure token storage powered by Android Keystore and iOS Keychain.

## Install

To use npm

```bash
npm install zbiolock
````

To use yarn

```bash
yarn add zbiolock
```

Sync native files

```bash
npx cap sync
```
